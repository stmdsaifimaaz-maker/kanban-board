# Kanban Board

A simple and interactive **Kanban Task Management Board** built using **React + Vite**.

This application allows users to manage tasks using a visual board with three stages: **Todo, In Progress, and Done**.

---

## Features

* Add new tasks
* Drag and drop tasks between columns
* Move tasks from **Todo → In Progress → Done**
* Delete tasks
* Edit tasks by double clicking
* Tasks saved using **Local Storage**
* Clean and responsive UI
* Smooth hover animations

---

## Tech Stack

* React
* Vite
* @hello-pangea/dnd (Drag & Drop Library)
* CSS

## Installation

Clone the repository:

```
git clone https://github.com/stmdsaifimaaz-maker/kanban-board
```

Go to the project folder:

```
cd kanban-board
```

Install dependencies:

```
npm install
```

Run the project:

```
npm run dev
```
